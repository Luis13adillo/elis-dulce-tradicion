### Add calculateDeliveryFee to ApiClient

**File:** `src/lib/api/index.ts`

Add a `calculateDeliveryFee` method to the `ApiClient` class. This method calls the Express backend endpoint `GET /api/v1/delivery/calculate-fee` and returns zone/fee data.

**Add after the `createPaymentIntent` method (around line 125) and before the closing `}`:**

```typescript
async calculateDeliveryFee(address: string, zipCode: string): Promise<{
  serviceable: boolean;
  fee: number;
  zone?: string;
  distance?: number;
  estimatedTime?: string;
}> {
  const API_BASE_URL = import.meta.env.VITE_API_URL || 'http://localhost:3001';
  try {
    const params = new URLSearchParams({ address, zipCode });
    const res = await fetch(
      `${API_BASE_URL}/api/v1/delivery/calculate-fee?${params.toString()}`,
      { credentials: 'include' } // Required for CSRF cookie cross-origin
    );
    if (!res.ok) return { serviceable: false, fee: 0 };
    return res.json();
  } catch {
    return { serviceable: false, fee: 0 };
  }
}
```

Note: This method is a GET request so it does NOT need a CSRF token header. `credentials: 'include'` is needed cross-origin to ensure the CSRF cookie is present if the browser needs it.
