    Because the Express backend (port 3001) is not started by playwright's webServer, all API calls must be intercepted with page.route(). This includes:
    - Pricing endpoints: `**/api/pricing/**` or `**/api/products**`
    - Order creation endpoint: `**/api/orders**`
    - Payment intent endpoint: `**/api/payments/create-payment-intent**` (or Supabase Edge Function equivalent)
    - Order status endpoint: `**/api/orders/**/status**`

    Stripe Elements loads in an iframe (stripe.com domain) — do NOT try to interact with the iframe directly. Instead, intercept the Stripe.js network calls via page.route() or use Stripe's test card numbers (4242 4242 4242 4242) if the iframe is accessible. More reliably: mock the `create-payment-intent` endpoint to return a fixture paymentIntentId, and mock the front-end's Stripe confirmation call via page.route('**/stripe.com/**', ...).

    **Structure (3 scenarios):**

    **beforeEach setup (shared):**
    ```typescript
    test.beforeEach(async ({ page }) => {
      // Mock all Express API calls — no backend needed
      await page.route('**/api/pricing**', route =>
        route.fulfill({ status: 200, contentType: 'application/json',
          body: JSON.stringify({
            cakePricing: [
              { size: 'small', base_price: 35 },
              { size: 'medium', base_price: 55 },
              { size: 'large', base_price: 75 },
            ],
            fillingPricing: [{ name: 'vanilla', additional_cost: 0 }],
            themePricing: [{ name: 'none', additional_cost: 0 }],
            deliveryZones: [],
            taxRates: [{ state: 'PA', county: null, rate: 0.06 }],
          })
        })
      );
      await page.route('**/api/orders**', route =>
        route.fulfill({ status: 201, contentType: 'application/json',
          body: JSON.stringify({ success: true, data: { id: 'test-order-123', status: 'pending' } })
        })
      );
      await page.route('**/create-payment-intent**', route =>
        route.fulfill({ status: 200, contentType: 'application/json',
          body: JSON.stringify({ clientSecret: 'pi_test_secret_mock' })
        })
      );
    });
    ```

    ```typescript
      }
    });
    ```

    **Scenario 2 — Stripe Checkout confirmation page renders with mocked payment intent:**
    ```typescript
    test('payment confirmation page renders with mocked payment intent', async ({ page }) => {
      // Mock Stripe.js network calls to avoid real payment processing
      await page.route('**stripe.com/v1/**', route =>
        route.fulfill({ status: 200, contentType: 'application/json',
          body: JSON.stringify({ id: 'pi_test_mock', status: 'succeeded', client_secret: 'pi_test_secret_mock' })
        })
      );

      // Navigate directly to PaymentCheckout with query params matching what Order.tsx would pass
      await page.goto('/payment?orderId=test-order-123&amount=5830&clientSecret=pi_test_secret_mock');

      // PaymentCheckout renders the Stripe form (StripeCheckoutForm wraps Stripe Elements)
      // The Stripe Elements iframe may not load without a real client secret, but the page should render
      await page.waitForSelector('body', { timeout: 10000 });
      await expect(page.locator('body')).toBeVisible();

      // The page should show the payment form container (not an error page or 404)
      // Accept either the Stripe iframe or the form wrapper as proof of render
      const formVisible = await page.locator('form, [data-testid="payment-form"], .payment-form, iframe[name*="stripe"]').isVisible().catch(() => false);
      const bodyText = await page.locator('body').innerText();
      // Page should not show a 404 or generic error
      expect(bodyText).not.toContain('404');
      expect(bodyText).not.toContain('Page not found');
    });
    ```

    ```typescript
      expect(bodyText.length).toBeGreaterThan(10);
    });
    ```
